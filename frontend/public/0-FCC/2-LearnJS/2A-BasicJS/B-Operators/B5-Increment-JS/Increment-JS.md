## FreeCodeCamp Information

- You can easily increment or add one to a variable with the ++ operator

``` JavaScript
// -+-+-+-+-+-+-+-+-
// Example: 
// -+-+-+-+-+-+-+-+-

/* 

i++;

is the equivalent of

i = i + 1;

*/


// -+-+-+-+-+-+-+-+-
// Directions: Change the code to use the ++ operator on myVar
// -+-+-+-+-+-+-+-+-

// Problem: var myVar = 87;

// myVar = myVar + 1;


// My solution:
myVar++;


```
